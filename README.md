# React MUI Boilerplate

React Boilerplate

## Includes

- Typescript
- ApolloClient
- codegen
- MaterialUI
- Recoil
- ESLint, Prettier, PreCommit
- Github Workflow

## How to try

Clone: `https://github.com/salman-ssi/react-mui-boilerplate-drops.git`
After cloning run below command:

    yarn && yarn start
